# 项目背景
基于Vue2+ElementUI组件库完成的后台文章数据管理项目
# 项目运行
```
# 克隆到本地
git clone https://github.com/GoodLuckyed/vue-event.git

# 安装依赖
npm install 或 yarn(推荐)

# 项目启动
npm run start
```
# 项目技术栈
1. vue2
2. elementUI
3. axios
4. vue-router
5. vuex
6. vue-cli
